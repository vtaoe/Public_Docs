
# COURSEENROLLSTATUS

code set for enrollment status in a course section

A codeset value indicating a student enrollment status in each course section in which they were enrolled.

## Codeset

These are options for this codeset:

|   CODE | Description   |
|--------|---------------|
|     01 | Enrolled      |
|     02 | Withdrawn     |
|     03 | Completed     |
|     04 | Incomplete    |
|     05 | Excused       |
