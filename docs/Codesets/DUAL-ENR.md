
# DUAL_ENR

Dual Enrollment

A codeset value indicating whether or not a student participated in Vermont's Dual Enrollment program at any point in the school year. Â 

## Codeset

These are options for this codeset:

| CODE   | Description                                                                                       |
|--------|---------------------------------------------------------------------------------------------------|
| N      | Student did not participate in Vermont's Dual Enrollment program at any point in the school year. |
| Y      | Student did participate in Vermont's Dual Enrollment program at some point in the school year.    |
